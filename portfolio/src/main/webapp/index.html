Projects

During my time so far at Carnegie Mellon University, I have had the wonderful opportunity to take courses covering a wide range of fascinating topics in the field of computer science. Many of these courses have culminated with a final project, where I have been able to gain a deeper understanding and learn the practical considerations of the concepts covered in the course by researching, designing, and implementing algorithms and analyses on real-world data and frameworks.

Spectral Clustering: A Python Implementation

Linear Algebra - Fall 2018  

Spectral clustering is an unsupervised learning algorithm that clusters data by performing dimensionality reduction before clustering the resulting lower-dimensional data. In this project, we used Python to reduce dimensionality by constructing a similarity matrix from our data and taking its top eigenvectors. We then implemented k-means clustering and ran it on the selected eigenvectors to produce clusters for the original data.

White Blood Cell Image Classification

Computational Biology - Spring 2019  

In this project, I attempted to create a computational model capable of classifying an image of a stained white blood cell into one of the four major types of white blood cells. Specifically, I used Python to preprocess a set of publicly available training images before using Tensorflow and Keras to create, train, and test three convolutional neural networks with the best model achieving an accuracy of 78%.

Range Analysis for Array Bounds Checking in Java

Program Analysis - Spring 2020  

In this project, I designed and implemented a context-sensitive interprocedural dataflow analysis that tracks the possible values of integer variables during program execution for Jimple -an intermediate representation of Java bytecode provided by the Soot framework. I utilized this analysis to provide errors and warnings about negative and out of bounds array accesses for Java code samples.
